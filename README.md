# ktx(WIP)

`ktx` enables you to add custom `contextMenu` to you application effortlessly.

planning to support:
- [ ] Vue
- [ ] React
- [ ] Angular
- [ ] Vanilla JS

[![NPM version](https://img.shields.io/npm/v/ktx?color=a1b858&label=)](https://www.npmjs.com/package/ktx)

## Installation

```bash
# npm
npm i -s ktx

# pnpm
pnpm add ktx

# yarn
yarn add ktx
```
